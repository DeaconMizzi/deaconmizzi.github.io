A custom-built, node-based quest and dialogue editor for Unity that enables designers to create dynamic NPC conversations, branching choices, and quest flows without writing code. Built for 2D pixel RPGs, it features state-based dialogue nodes, condition-driven transitions, and support for quest offering, tracking, and turn-in events.

This tool supports:

  • Branching dialogue trees with choice nodes
  • Quest state control (offer, in progress, turn-in)
  • Speaker assignment and speech bubble style selection
  • In-editor previewing of dialogue sequences and flow connections

Designed to streamline narrative design in Unity projects, it allows non-programmers to visually manage quests and dialogue with intuitive drag-and-drop functionality.
