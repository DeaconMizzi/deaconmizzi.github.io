Description

The Infinite Tales website is a digital bookstore platform designed to make discovering and managing books simple and engaging. It features a wide collection of titles across various genres, allowing users to browse, search, and explore new reads through a clean, user-friendly interface. Visitors can rate books based on their reading experience, helping others find top-rated content. A personalized wishlist feature lets users bookmark titles they’re interested in, making it easy to track future reads.

Challenges & Future Improvements

This was my first project using PHP, so getting started with server-side scripting and database integration came with a learning curve. One of the more persistent challenges was managing the layout and sizing of the book cards across different screen sizes, which required several iterations to get right.

Implementing the rating system also presented difficulties, as early versions had a few bugs that affected user input and data consistency. Working with the database was another hurdle, but through this project, I was able to grasp the fundamentals of handling queries and storing user-generated content.

Looking ahead, there are several areas I'd like to improve, including enhancing the user interface with better color choices, refining the layout structure, and introducing more dynamic features such as user reviews, filters, or animations to elevate the experience.
