Description

The Last Ember is a 2D pixel art action-adventure game developed as part of my thesis on immersive narrative design. Inspired by games like Hyper Light Drifter and The Wizard Legend, it features a branching story where player choices shape both abilities and the world itself. Across three distinct regions, each tied to a thematic Ember, players face enemies, solve moral dilemmas, and decide the fate of a crumbling land. Each Ember offers a new power and changes the environment in meaningful ways, leading to multiple endings based on the player’s path.

Challenges and Insights

One major challenge was building a flexible narrative system that could track player decisions and trigger dynamic changes across the world. This required custom memory flags and a modular quest/dialogue system that could update the game state seamlessly. Another key area was implementing responsive combat and abilities like the Bloomstep dash and FlameGuard shield, each with visual feedback and world interactions. Testing and iteration were essential, especially when balancing abilities and ensuring clear player feedback. The Last Ember is a prototype with multiple endings and a polished narrative flow. It has been a passion project that combined storytelling, systems design, and player agency, core elements of what I aim to explore further in future projects.
